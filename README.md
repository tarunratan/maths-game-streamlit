## Features

  - **MCQ Mode**: Answer multiple-choice questions.
  - **Manual Input Mode**: Type your answer manually.
- **Difficulty Levels**: Easy, Medium, and Hard.
- **Dynamic Timer**: 10 seconds per question.
- **Interactive Charts**: Visualize your response times with bar and line charts.
- **Progress Tracking**: Track your progress with a progress bar.
- **Dark/Light Theme**: Choose your preferred theme.

## How to Run Locally

1. **Clone the repository**:
   ```bash
   git clone https://github.com/tarunratan/maths-game-streamlit.git
   cd maths-game-streamlit
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the app**:
   ```bash
   streamlit run app.py
   ```

4. Open your browser and navigate to `http://localhost:8501`.
